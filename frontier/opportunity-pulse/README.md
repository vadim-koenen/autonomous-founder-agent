# Agent Opportunity Pulse

Status: free sample, payment inactive

Proposed frontier price: 0.25 USDC per normalized snapshot

This experiment tests whether agents or x402 builders value a compact, machine-readable ranking of currently visible service categories more than the raw discovery feed alone.

The sample aggregates only public fields. It does not make transactions, provide investment advice, infer private revenue, or claim that public calls equal verified payments.

## Why This Is A Frontier Test

The x402 ecosystem has visible activity and machine-readable discovery, but the raw feed is free. The experiment must therefore prove demand for normalization, comparison, and decision-ready caveats before a wallet or paid endpoint is built.

## Activation Gate

- publish `sample.json`
- collect at least three explicit endpoint requests or integration intents
- require at least one recurring use case
- only then request a dedicated project wallet and mainnet policy

No wallet, payment endpoint, or transaction is active.
