# Agent Launch Gate

Status: validation sample

Proposed founding license: $39

Agent Launch Gate is a compact, platform-independent set of test cases and report rules for indie teams preparing an agent, MCP server, or LLM workflow for launch.

The paid product is not on sale. The repository currently publishes only a five-case sample and report schema so demand can be measured before checkout or private fulfillment infrastructure is built.

## Intended Outcome

A buyer should be able to run a consistent launch decision instead of relying on an unstructured chat session or a generic checklist.

## Sample Contents

- `free-sample/cases.sample.json`: five representative cases
- `free-sample/report.schema.json`: machine-readable report contract

## Full Draft Scope

- 25 cases
- scoring and severity rules
- regression rerun checklist
- report template
- launch decision rubric
- adaptation notes for HTTP, MCP, and task-based agents
