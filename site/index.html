What The Kit Packages

Product Metadata

Agent Policy

Seller Checklist

A fast QA pass so the public page and machine-readable product card do not contradict each other.

Launch Boundary

  • No payment is enabled from this static page until a human-provided public checkout URL is configured.
  • No NFT is minted from this project.
  • No email, ad, public post, wallet transaction, or trading action is performed by the code.
  • The paid bundle is prepared as a draft until an approved public checkout URL is connected.
